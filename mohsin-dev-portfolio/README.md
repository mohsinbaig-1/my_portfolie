# Mohsin Baig — Portfolio

Production-ready portfolio for **Mohsin Baig**, Software Engineering student and Unreal Engine 5 developer.

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- Framer Motion
- GitHub API integration

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Project Structure

```
src/
├── app/           # Routes, layout, API
├── components/    # UI, layout, sections
├── data/          # Site content
├── hooks/         # Scroll & navigation hooks
├── lib/           # Utilities & GitHub fetch
└── types/         # TypeScript types
```

## Resume

Add your PDF to `public/resume/mohsin-baig-resume.pdf` for the download button.

## Deploy

Deploy to Vercel, Netlify, or any Node.js host:

```bash
npm run build
npm start
```
